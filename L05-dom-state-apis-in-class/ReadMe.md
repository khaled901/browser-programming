# L05 — DOM • State • Browser APIs

This project contains exercises 1–6 for Lecture 5.

## Topics Covered

* DOM manipulation (input, events, event delegation)
* State-driven UI (counter, mini cart)
* Browser APIs (localStorage, Geolocation)

## Features

* Live character counter
* Toggle list with add/remove
* Counter using state → render pattern
* Mini shopping cart with total calculation
* Notes saved in localStorage
* Get user location using Geolocation API

## Notes

* No external libraries used
* All features work in the browser
* Geolocation requires localhost or HTTPS
